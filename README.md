# Consumo de API - Pokémon

Este repositório demonstra um projeto prático de consumo de API usando JavaScript, HTML e CSS. O projeto utiliza a PokeAPI para buscar e exibir informações sobre Pokémon de forma interativa.

## Objetivo

- Demonstrar como fazer requisições HTTP para APIs externas.
- Manipular dados JSON retornados pela API.
- Exibir informações dinamicamente na página web.
- Praticar conceitos de JavaScript assíncrono (promises e fetch).
- Servir como exemplo de integração de API em aplicações web simples.

## Conteúdo do Projeto

### Funcionalidades Implementadas

1. **Lista de Pokémon**
   - Carrega e exibe os primeiros 10 Pokémon da API.
   - Mostra imagem e nome de cada Pokémon.

2. **Pokémon Aleatório**
   - Gera um ID aleatório e busca um Pokémon específico.
   - Exibe nome, imagem, altura e peso.

3. **Pokémon por Tipo**
   - Permite selecionar um tipo de Pokémon (ex: fogo, água, grama).
   - Carrega e exibe até 10 Pokémon daquele tipo.

### Arquivos do Projeto

- `index.html`: Estrutura da página web com seções para cada funcionalidade.
- `script.js`: Lógica JavaScript para consumir a API e manipular o DOM.
- `styles.css`: Estilos CSS para uma interface limpa e responsiva.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização e layout responsivo.
- **JavaScript (ES6+)**: Lógica de consumo da API e manipulação do DOM.
- **PokeAPI**: API externa para dados de Pokémon (https://pokeapi.co/).

## Como Usar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em um navegador web moderno.
3. Clique nos botões para carregar as diferentes seções de Pokémon.
4. Para a seção "Pokémon por Tipo", selecione um tipo no dropdown antes de clicar em "Carregar por Tipo".

Este projeto ilustra conceitos fundamentais de desenvolvimento web moderno, focando em integração com APIs externas e manipulação dinâmica de conteúdo.
